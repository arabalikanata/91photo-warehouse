# 91photo-warehouse
存储91photos网站图片的仓库
审核员操作方法：
1:从"uploads"文件夹选择通过审核的图片；
2:点击右上角的铅笔工具。
3:点击"Cancel changes"左侧的文本框。
4:一直按空格，直至路径名中的"uploads"消失。
5:在同一个文本框中输入"Approved Photos"，注意区分大小写。
6:输入"/"，将文件移入"Approved Photos"文件夹。
7:点击右侧绿色"Commit Changes"按钮作出更改。如忽略此步骤，之前的操作全部作废。
8:将"Commit message"文本框中的文字全部删除。
9:点击弹窗右下角的"Commit Changes"按钮。
10:至此，所有操作均已完成。
